# 💰 BucksFlow — Smart Money Tracker

Free, no-signup personal finance tracker. Track income/expenses, visualize spending with charts, all saved locally.

**🔗 [chanmin713.github.io/bucksflow](https://chanmin713.github.io/bucksflow/)**

## Features

- 📊 Income & expense tracking with visual charts
- 🏷️ Category breakdown (Food, Transport, Housing, etc.)
- 💾 localStorage — no account needed, no data leaves your browser
- 🌙 Dark mode UI, mobile responsive
- 🇰🇷 Toss / KakaoPay / PayPal donation support

## Tech Stack

Vanilla HTML + CSS + JavaScript. Zero dependencies, no frameworks.
